一、项目介绍
项目名称：

华夏命名

当前模块：

汉魏古法命名

定位：

借鉴东汉至三国时期士人的命名传统，为现代用户提供符合古典文化审美的"名 + 字"命名方案。后续可扩展传统命名、唐宋命名、诗经命名、八字命名等模块。

本项目遵循：

名以立志，字以明德。

所有名字均采用规则引擎生成。

AI仅用于解释。
